\hypertarget{index_CARATTERISTICHE}{}\doxysection{SOFTWARE}\label{index_CARATTERISTICHE}
Il programma consente la gestione di una biblioteca tramite un menu dove puoi scegliere cosa fare\+:
\begin{DoxyItemize}
\item inserimento di uno o piu libri
\item stampare il catalogo dei libri
\item rimuovere un libro dal catalogo tramite ISBN
\item ricercare un libro dal catalogo tramite ISBN
\item modificare dati di un libro dal catalogo tramite ISBN
\item salvataggio del catalogo su un file .txt
\end{DoxyItemize}


\begin{DoxyParams}{Parameters}
